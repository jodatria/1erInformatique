🔍 1. Vocabulaire de base

Voici les mots essentiels pour bien commencer :

  • Fichier : ensemble d’informations enregistrées (image, texte, vidéo...)
  • Dossier : conteneur pour organiser plusieurs fichiers
  • Périphérique : appareil connecté à l’ordinateur (clavier, souris, imprimante...)
  • Extension : suffixe du nom de fichier qui indique son type (.jpg, .pdf, .docx)

🗂️ 2. Gestion des fichiers

Un bon rangement aide à retrouver facilement ses documents.

  • Créer un dossier pour chaque projet ou cours
  • Renommer les fichiers avec des noms clairs (ex: "devoir_maths.pdf")
  • Classer par date, matière ou type de document
  • Exporter un document signifie le sauvegarder dans un autre format

Voici une idée d’organisation :

    • fichier : exercice_fichier.pdf
