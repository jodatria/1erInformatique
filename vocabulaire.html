🔍 Vocabulaire de base

Voici quelques définitions importantes :

  • Fichier : ensemble d’informations enregistrées sur un support numérique (ex: image, document...)
  • Dossier : conteneur qui regroupe plusieurs fichiers ou d'autres dossiers.
  • Périphérique : élément connecté à l’ordinateur (clavier, souris, imprimante...)
  • Extension : suffixe du nom d’un fichier indiquant son type (ex: .jpg, .pdf, .docx).
